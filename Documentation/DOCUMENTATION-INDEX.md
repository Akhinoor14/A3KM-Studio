## 📝 Documentation Standards

### File Naming:
- Lowercase with hyphens: `mobile-system-complete.md`
- Descriptive names: `github-sync-complete.md`
- Category prefix in folder: `01-website-overview/`

### Frontmatter Template:
```yaml
---
title: "Document Title"
date: 2026-02-12
author: Md Akhinoor Islam
organization: A3KM Studio | Noor Academy
category: category-name
tags: [tag1, tag2, tag3]
---
```

### Content Structure:
1. Heading 1: Main title
2. Brief introduction (কী এটা?)
3. Sections with Heading 2
4. Sub-sections with Heading 3-4
5. Code examples যেখানে দরকার
6. End with: "শেষ Update:" date

### Writing Style:
- **Natural Bangla** - AI-generated মনে হবে না
- Technical terms English letterএ (GitHub, API, token)
- Casual tone - "আমি", "তুমি" use করা যায়
- Real examples - কোন fake/placeholder data নয়
- Code snippets actual codebase থেকে
