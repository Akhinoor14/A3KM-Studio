Security System

9-Layer Security

Login এর পর থেকে logout পর্যন্ত — প্রতিটি স্তরে session protect করা হয়।

LAYER 01
SHA-256 Password Hash
Password plain text-এ কোথাও নেই। Web Crypto API দিয়ে hash করে compare। crypto.subtle.digest('SHA-256')
LAYER 02
Browser Fingerprint
Canvas fingerprint + User-Agent + timezone + screen dimensions — এগুলো combine করে unique device ID তৈরি করে।
LAYER 03
30-Minute Session Timer
Login-এর পর countdown শুরু। 30 মিনিট নিষ্ক্রিয় থাকলে auto logout + sessionStorage clear।
LAYER 04
DevTools Detection
Window resize ratio দিয়ে DevTools open detect। খুললে instant logout + warning message।
LAYER 05
Console Tampering Guard
console.log override করে যেকোনো JS console access monitor করে। Attempt log হয়।
LAYER 06
Right-Click Disabled
Dashboard page-এ right-click context menu block। Inspect Element shortcut (F12, Ctrl+Shift+I) intercept।
LAYER 07
Wrong Attempt Counter
৩ বার wrong password দিলে page lock → 60 সেকেন্ড cooldown। Attempt count localStorage-এ।
LAYER 08
Session Validation
প্রতিটি dashboard page load-এ validateSession() চলে — invalid হলে auth page-এ redirect।
LAYER 09
No Backend Dependency
সব security client-side — server crash হলেও portal নিরাপদ থাকে। Data GitHub/localStorage-এ।

File Structure

Portal Structure

File / FolderPurpose
Only-boss/auth/index.htmlLogin page — crown trigger redirects here. SHA-256 check happens here.
Only-boss/auth/auth.jsAll authentication logic: hash compare, fingerprint, attempt counter, lockout.
Only-boss/dashboard/index.htmlMain dashboard after login — session timer visible, 8 tool cards.

Security Note
এই portal সম্পূর্ণ client-side। Browser Developer Tools-এ সব code দেখা যায় — তবে SHA-256 hash ছাড়া password জানা সম্ভব নয়। Production-এ backend auth যোগ করা recommended।
