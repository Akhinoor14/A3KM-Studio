Only-Boss
Complete Admin Guide

Only-Boss is the admin control system for A3KM Studio — protected login, unified dashboard, and 20+ specialized management tools for content, projects, settings, and security.

Only-Boss হলো A3KM Studio-র সম্পূর্ণ admin system। Password দিয়ে login করো, তারপর content manage করো, projects update করো, settings configure করো।

Access

How to Enter

Only-Boss-এ ঢোকার দুটো উপায়।

Desktop Secret Button
Desktop home page-এ hidden admin button। Console access guard bypass করে থাকে। → Only-boss/auth/only-boss.html
Mobile 10-Tap Trigger
Mobile home page-এ specific element 10 বার tap করলে mobileSecretTap() trigger → admin login page redirect।
Authentication

Login Security Flow

Multi-layer authentication system। — auth/only-boss.html

1
Password Entry
Password field with show/hide toggle। autocomplete="off" — no browser autofill। Plain text → SHA-256 hash comparison।
2
SHA-256 Hash Comparison
Entered password hash করে stored expected hash-এর সাথে compare করে। Hash generation tool: managers/security/generate-password-hash.html
3
Session Storage
sessionStorage.setItem('onlyBossAuthenticated', ...) + authTime সেট হয়। Tab বন্ধ হলে session clear — auto logout।
4
Dashboard Redirect
Successful auth → only-boss-dashboard-redesigned.html-এ redirect। প্রতিটা dashboard page-এ auth check।
5
Auto Logout
Home link click → sessionStorage.removeItem('onlyBossAuthenticated')। Browser tab close → session clear। Re-login required।

Main Tool

Command Center

Unified control hub — 3 tabs-এ organized। — managers/shared/command-center.html

Command Center is the primary admin hub. Dashboard card click → Command Center। এখান থেকে API setup, content manage, activity দেখা — সব possible।
সব major admin কাজ এখানেই হয়।
Tab 1: API Configuration
Tab 2: System Tools
Tab 3: Activity Log
API Configuration Tab Features
  • GitHub Personal Access Token — save, clear, test connection
  • Live API Call Monitor — real-time: last URL, time, status, rate limit, response duration
  • API Call History — chronological log stream
  • QR Code generate → scan from phone (token sharing)
  • API Status Bar — GitHub connection, rate limit remaining, uploader status, content count
Tab 2: System Tools → Quick links to all managers। Tab 3: Activity Log → recent admin actions।

Security

Security Tools

Admin password এবং security management।

SHA-256 Hash Generator
Type করো → real-time SHA-256 hash generate। New admin password set করতে হলে এখানে hash বানিয়ে auth code-এ paste করো। security/generate-password-hash.html
Session-Based Auth
sessionStorage — browser tab বন্ধ হলেই session expire। _vk verification key + _sid session ID hash — double validation।
Console Guard
Desktop site-এ devtools/console guard active। F12 / right-click context menu disable on public pages।
QR Token Transfer
GitHub API token QR code-এ encode করো → phone-এ scan। Cross-device token sharing without copy-paste।
