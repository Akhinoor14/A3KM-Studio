   - SolidWorks Modeling Quick Reference
   - Status: ⏳ To be added

## How to Add PDFs:

1. Place your PDF file in this folder
2. Make sure the filename matches exactly with the `downloadUrl` in `books.json`
3. Recommended file size: Under 15 MB for faster downloads
4. Use descriptive filenames with hyphens (e.g., `arduino-handbook-bn.pdf`)

## Note:
The system will work without these files - users will see a "Coming Soon" message or download button will be disabled until PDFs are added.
