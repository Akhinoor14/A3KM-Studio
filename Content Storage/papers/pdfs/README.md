## How to Add Research Papers:

1. Place your PDF file in this folder
2. Make sure the filename matches exactly with the `pdfUrl` in `papers.json`
3. Include proper citations and references in your paper
4. Use descriptive filenames (e.g., `solar-optimization-bd.pdf`)

## Paper Format Guidelines:

- **File Format:** PDF only
- **File Size:** Preferably under 10 MB
- **Paper Structure:** 
  - Title page with authors and institution
  - Abstract
  - Keywords
  - Introduction
  - Methodology
  - Results
  - Discussion
  - Conclusion
  - References

## Note:
The system will work without these files - users can still view paper metadata, abstract, and keywords. The "View Paper" button will indicate if PDF is available.
