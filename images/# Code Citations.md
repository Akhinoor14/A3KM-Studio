```

{{request_id}}     — Firestore document ID
```

EmailJS template এ **Subject** এবং **Body (HTML)** দিতে হয়। এখানে পুরো HTML template দিচ্ছি — EmailJS dashboard এ গিয়ে `template_wxyrgg7` edit করে এটা paste করো:

---

**Subject field:**
```
✅ Access Granted — A3KM Studio | {{to_name}}
```

---

**Body (HTML) field — পুরোটা:**

```html
<!DOCTYPE html>
<html lang="en">
<head>
<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<title>Access Granted — A3KM Studio</title>
```
