# App Icons (PWA)

Use the same website icon as the main app icon. Place the following PNG files in this folder:

- icon-180.png  (iOS Home Screen / Apple touch icon)
- icon-192.png  (Android PWA install)
- icon-512.png  (PWA install large)

Free ways to generate from your existing favicon.svg (or any logo):

1) PWABuilder Images (free, no login)
   - https://www.pwabuilder.com/imageGenerator
   - Upload favicon.svg → download the generated assets → pick the files above and copy here.

2) Favicon.io (free)
   - https://favicon.io/favicon-converter/
   - Upload your SVG/PNG → download PNGs at 180/192/512 → rename as above and copy here.

3) RealFaviconGenerator (free tier)
   - https://realfavicongenerator.net/
   - Upload your SVG/PNG → download package → copy the 180/192/512 PNGs here.

Notes
- Keep the icon background solid and high contrast (A3KM red/black/white works great).
- Maintain safe padding so the A3KM mark isn't cropped.
- Filenames must match exactly (icon-180.png, icon-192.png, icon-512.png).
