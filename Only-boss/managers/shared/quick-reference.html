API Configuration

  • GitHub Token: Required for uploads
  • Generate at: github.com/settings/tokens
  • YouTube API: Optional (for duration fetch)
  • Test connection after saving

Manager Operations

  • View: Grid view with all content cards
  • Edit: Click Edit button ‚Üí Modal opens ‚Üí Save
  • Delete: Confirmation required ‚Üí Removes from GitHub
  • Stats: View total counts & categories
Open Content Hub

Backend Functions

  • loadContent(type) - Load all content
  • editContent(type, id, data) - Update
  • deleteContent(type, id) - Remove
  • uploadFile(path, content) - GitHub upload
View Integration

Tips & Tricks

  • Auto-fill: Categories load automatically from JSON
  • Tags: Press Enter to add, click X to remove
  • Cover: Auto-generated if not provided
  • Video ID: Auto-extracted from YouTube URL

Troubleshooting

  • Upload fails: Check GitHub token configuration
  • Categories missing: Verify JSON files exist
  • Edit not saving: Check network & token validity
  • API warning: Go to API Config to add token
