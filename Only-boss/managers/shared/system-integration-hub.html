Complete Data Flow

1. User opens Dashboard → Clicks Content Upload
↓
2. upload-interface-new.html → Select content type (5 cards)
↓
3. Individual upload form (posts/videos/books/papers/educational)
↓
4. Form validates → Loads categories from JSON
↓
5. User fills form → Uploads file/enters data
↓
6. content-manager.js processes data
↓
7. github-content-uploader.js (auto-loads token from localStorage)
↓
8. Uploads to GitHub Repository (Content Studio/)
↓
9. Updates JSON file with new content metadata
↓
10. Manager pages (posts/videos/books-manager.html) display content
↓
11. Edit/Delete operations via manager → Updates GitHub
