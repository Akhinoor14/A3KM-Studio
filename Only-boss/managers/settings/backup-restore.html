ব্যবহার গাইড / Usage Guide

Backup & Restore System এর সম্পূর্ণ ব্যবহার নির্দেশিকা

🎯 মূল উদ্দেশ্য / Original Purpose

Bangla: এটি আপনার সম্পূর্ণ website data (content, settings, media, projects) এর secure backup তৈরি করে এবং যেকোনো সময় restore করার সুবিধা দেয়।

English: Secure your website data with automatic backups and restore functionality. Create complete backups or select specific data categories.

দ্রুত শুরু / Quick Start

Bangla - Backup তৈরি:

  • Backup Name লিখুন (optional)
  • যে data backup করবেন সেগুলো checkbox দিয়ে select করুন
  • "Create Backup" button এ click করুন
  • Backup automatically save হয়ে যাবে

English - Restore: Select backup from history → Click "Restore" → Confirm → Data restored!

Backup Types

Bangla:

  • Full Backup: সব checkboxes select করুন - সম্পূর্ণ website data backup হবে
  • Selective Backup: শুধু নির্দিষ্ট categories select করুন (e.g., শুধু Content বা Settings)
  • Quick Backup: সব checkbox checked রেখে সরাসরি "Create Backup" click করুন

Backup Categories:

  • 📚 Content (Books, Videos, Papers, Posts)
  • ⚙️ Site Settings (Colors, API keys, configurations)
  • 🖼️ Media Library (Images, PDFs, documents)
  • 🎓 Certificates (Academic, Skill, Medical)
  • 🔧 Projects (Arduino, MATLAB, SolidWorks)
  • 📊 Analytics Data (Statistics, activity logs)

Backup History Management

Bangla:

  • View Backups: "Backup History" section এ সব previous backups দেখবেন
  • Restore: যেকোনো backup এর "Restore" button click করুন
  • Download: "Download" button এ click করে JSON file হিসেবে save করুন
  • Delete: পুরোনো backup delete করতে "Delete" button ব্যবহার করুন
  • Clear All: সব backup history clear করতে "Clear All Backup History" button

English: Each backup shows date, size, and included data categories.

Restore Process

Bangla - কীভাবে Restore করবেন:

  • Method 1 (From History): Backup History থেকে backup select করে "Restore" click করুন
  • Method 2 (From File): "Restore from File" section এ JSON file upload করুন
  • ⚠️ Warning: Restore করলে current data replace হয়ে যাবে
  • 💡 Pro Tip: Restore করার আগে নতুন backup তৈরি করুন

Restore Steps:

  • 1️⃣ Create current data backup (safety)
  • 2️⃣ Select backup to restore
  • 3️⃣ Click "Restore" and confirm
  • 4️⃣ Wait for success message
  • 5️⃣ Refresh page to see restored data

Export Individual Data

Bangla:

  • "Export Data" section এ যান
  • যে data export করবেন সেই button click করুন (Books, Videos, Settings, etc.)
  • JSON file download হয়ে যাবে
  • এই files শুধু একটি specific category এর data contain করে

Use Case: যখন শুধু specific data (e.g., শুধু Books অথবা শুধু Settings) backup/share করতে চান।

Backup Comparison

Bangla:

  • দুটি backup select করুন dropdown থেকে
  • "Compare Backups" button click করুন
  • System automatically দেখাবে কী data changed হয়েছে
  • এটি দিয়ে progress track করতে পারবেন (কতটা content add হয়েছে)

English: Compare two backups to see what data was added, modified, or removed.

প্রো টিপস / Pro Tips

  • 📆 প্রতি সপ্তাহে একবার full backup করুন
  • 💾 Important backups external storage এ save করুন
  • 📝 Backup name এ date এবং purpose লিখুন (e.g., "Before Major Update - 2024-01-15")
  • 🔄 Restore করার আগে সবসময় current data backup করুন
  • 🗂️ পুরোনো unnecessary backups regularly delete করুন
  • ☁️ Backups Google Drive/Dropbox এ sync করে রাখুন
  • 🔒 Backup files secure location এ রাখুন (password-protected folder)

সমস্যা সমাধান / Troubleshooting

Backup তৈরি হচ্ছে না?

  • Browser console (F12) খুলে error check করুন
  • LocalStorage space পর্যাপ্ত আছে কিনা verify করুন
  • অন্তত একটি checkbox select করেছেন কিনা check করুন

Restore কাজ করছে না?

  • Backup file টি corrupt নেই কিনা check করুন
  • JSON format valid কিনা verify করুন
  • Page refresh করে আবার try করুন

Backup file খুব বড়?

  • Selective backup করুন (only necessary data)
  • পুরোনো unused content delete করুন
  • Media files compress করুন
