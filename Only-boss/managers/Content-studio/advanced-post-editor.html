How To Use

Creating Tables
  1. Click the Table button (blue icon)
  2. Set number of rows (1-20) and columns (1-10)
  3. Check "Include Header Row" if needed
  4. Watch the live preview
  5. Click "Insert Table"
  6. Fill in your data manually in the markdown
Adding Images
Method 1: Upload
  1. Click Image button → Upload from Device tab
  2. Click to select or drag & drop images
  3. Preview shows, click "Insert Image(s)"
  4. Images embedded as Base64 (no external hosting needed)
Method 2: URL
  1. Switch to Insert from URL tab
  2. Paste image URL
  3. Add alt text (optional)
  4. Click "Preview URL" to verify
  5. Click "Insert Image"
Embedding Videos
  1. Click the Video button
  2. Paste YouTube or Vimeo URL
  3. Iframe code auto-generated and inserted
  4. Video will render in preview and published post
Using Emojis
  1. Click the Emoji button
  2. Select category: Faces, Symbols, Objects, Flags
  3. Click any emoji to insert at cursor position
  4. Modal closes automatically

Best Practices

✅ DO
  • Use H1 only for title
  • Preview before publishing
  • Add alt text to images
  • Keep tables under 10 rows
  • Use callouts for important notes
❌ DON'T
  • Upload images over 5MB
  • Use multiple H1 headings
  • Leave empty headings
  • Nest tables inside tables
  • Write very long paragraphs
