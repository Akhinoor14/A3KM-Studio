📘 User Guide - How to Use This Manager

1. Upload Research Papers

  • Click "📤 Upload Paper" tab from the sidebar
  • Fill in paper details: Title, Authors, Institution, Abstract
  • Select Category & Subcategory from hierarchical dropdown
  • Add keywords, publication year, journal name
  • Upload PDF file (optional: DOI link, citations count)
  • Use Rich Text Editor for formatting abstract with headings, lists, and quotes
  • Click Template Buttons (Research/Review/Survey) for pre-formatted abstract structures

2. Manage Your Collection

  • Go to "📚 Manage Papers" tab to view all papers
  • Search & Filter: Use search bar, category, year, language, date range filters
  • Sort Options: Sort by newest/oldest, title, most cited, most downloaded
  • Bulk Operations: Select multiple papers using checkboxes
  • Click "✅ Select All" to select all visible papers
  • Use "🗑️ Delete Selected" to remove multiple papers at once
  • Edit/View: Click on any paper card to edit details or view full information
  • Download PDF: Click PDF icon on paper card to download

3. View Statistics & Analytics

  • Navigate to "📈 Statistics" tab for detailed analytics
  • Papers by Category: See distribution across research fields
  • Papers by Year: Track publications timeline
  • Language Breakdown: View papers in English, বাংলা, हिंदी
  • Top Authors: Most productive authors based on paper count
  • Most Cited: Highest impact papers by citations
  • Statistics auto-update when you add/edit/delete papers

4. Advanced: JSON Editor

  • ⚠️ Advanced Users Only: Direct editing of papers.json database
  • Click "🔧 JSON Editor" tab to access
  • Validate: Check JSON syntax before saving (click ✓ Validate JSON)
  • Format: Auto-format JSON with proper indentation
  • Backup: Always create backup before manual edits
  • Edit category groups, subcategories, or paper metadata
  • Reload: Discard changes and reload original data
  • Click "💾 Save Changes" when done (validates automatically)

💡 Pro Tips & Best Practices

  • Rich Abstracts: Use formatting (bold, italic, headings) for better readability
  • Keywords: Add 3-7 relevant keywords for better searchability
  • DOI Links: Include DOI for academic credibility and citation tracking
  • PDF Organization: Store PDFs in Content Storage/research-papers/ folder
  • Consistent Naming: Use format: Author_Year_Title.pdf
  • Regular Updates: Keep citation counts and download stats current
  • Backup: Periodically save papers.json file externally
  • Categories: Choose most specific subcategory for accurate organization
