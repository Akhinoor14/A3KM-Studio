Instructions
  1. Make sure backend server is running (python secure-proxy-server.py)
  2. Generate GitHub tokens from: GitHub Settings
  3. Enter your admin password (set in backend .env file)
  4. Paste tokens (one per line) and click "Add Tokens"
  5. Tokens are encrypted and stored securely on backend
  6. Frontend automatically uses backend proxy (no user action needed)
