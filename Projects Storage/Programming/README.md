# Programming Storage

Code files for the [Programming & DSA](../../Projects%20Code/programming/programming-listing.html) section.

## Folder Structure

```
Programming/
├── python/
│   ├── problem-solving/
│   │   ├── arrays-strings/        ← Array, string manipulation problems
│   │   ├── dynamic-programming/   ← DP, memoization, tabulation
│   │   ├── linked-lists/          ← Linked list operations
│   │   ├── trees-graphs/          ← BFS, DFS, tree traversal
│   │   ├── binary-search/         ← Binary search patterns
│   │   └── sorting-searching/     ← Sorting algorithms
│   ├── data-structures/           ← Custom DS implementations
│   ├── algorithms/                ← Algorithm implementations
│   ├── mini-projects/             ← Small Python projects
│   └── automation/                ← Automation scripts
├── javascript/
│   ├── problem-solving/           ← JS algorithm problems
│   └── mini-projects/             ← JS mini apps
└── cpp/
    ├── problem-solving/           ← C++ competitive programming
    └── algorithms/                ← C++ algorithm implementations
```

## File Naming Convention

`problem-title-kebab-case.py` — e.g., `two-sum.py`, `maximum-subarray.py`
