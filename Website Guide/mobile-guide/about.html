About Page Features

Desktop about-এর সব features mobile-এ touch-friendly pattern-এ adapt করা হয়েছে।

Education Timeline
Accordion-style cards for BSc KUET, HSC, SSC. Tap header to expand full details. max-height transition creates smooth collapse effect.
Tap করলে education detail expand হয়। CSS max-height: 0 → auto transition।
Skill Chips
Horizontally scrollable row of 12 skill chips. CSS overflow-x: scroll; scroll-snap-type: x. Momentum scrolling with -webkit-overflow-scrolling: touch.
Swipe করে chips scroll করা যায়। Touch momentum scrolling enabled।
Club Cards
5 collapsible club/organization cards. Click header → toggleClubDetails(n) JS function → expand description + role info.
৫টি club — tap করলে expand, আবার tap করলে collapse। JS accordion pattern।
CEFR Language
Visual CEFR language proficiency bars for English and Bangla. Same as desktop but single-column layout on mobile.
English C1, Bangla native — visual bar chart। Mobile-এ full-width, desktop-এ grid।
