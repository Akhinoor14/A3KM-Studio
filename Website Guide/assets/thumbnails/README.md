# 🖼️ Documentation Thumbnails

এই folder এ documentation thumbnails রাখা হয়।

## Thumbnail Specifications:

- **Size:** 400x250px (16:10 ratio)
- **Format:** JPG or PNG
- **File size:** < 100KB (optimized)
- **Naming:** lowercase-with-hyphens.jpg

## Naming Convention:

```
category-topic.jpg
```

**Examples:**
- `github-sync-implementation.jpg`
- `mobile-css-guide.jpg`
- `only-boss-token-system.jpg`
- `content-studio-upload.jpg`

## How to Create Thumbnails:

### Option 1: Gradient Background
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
+ Category Icon (large)
+ Title Text (white)
```

### Option 2: Screenshot
- Documentation এর screenshot নিন
- Crop to 400x250px
- Add subtle overlay

### Option 3: Custom Design
- Figma/Photoshop ব্যবহার করুন
- Category color theme follow করুন
- Brand consistency maintain করুন
