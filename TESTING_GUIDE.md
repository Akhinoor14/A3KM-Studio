## âŒ Common Errors & Solutions

### Error: "GitHub token not found"
**Solution:** Go back to token setup, paste token, click Validate

### Error: "Failed to upload"
**Solutions:**
- Check internet connection
- Verify token has 'repo' scope
- Check file size < 100MB
- Try again (might be temporary network issue)

### Error: "Invalid file type"
**Solution:** Only use allowed extensions:
- 3D: .glb, .gltf
- CAD: .sldprt, .sldasm, .slddrw
- Docs: .pdf, .txt, .md
- Images: .png, .jpg, .jpeg

### Error: Model not showing in gallery
**Solutions:**
- Refresh page (Ctrl+F5)
- Check GitHub - files uploaded?
- Clear browser cache
- Check model number matches

### Error: 3D viewer shows "Not Available"
**Solutions:**
- Check GLB file exists in GitHub folder
- Verify filename: `Model XX Basic.glb` (exact pattern)
- Check file is valid GLB (not corrupted)
- Try renaming to `model.glb` (lowercase)

### Error: ZIP download fails
**Solutions:**
- Check internet connection
- Try again (GitHub API might be slow)
- Download files manually from GitHub
- Check browser allows downloads
