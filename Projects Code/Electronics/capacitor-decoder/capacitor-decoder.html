Complete Usage Guide

How to Read Different Capacitor Markings

1. Ceramic Disc Capacitors (3-Digit Code)

Look for 3 digits printed on the body (e.g., 104, 223, 475)
Optional: May have tolerance letter after code (104K, 223J)
Use the 3-digit decoder section of this tool
Result is always in picofarads (pF), convert to nF or ¬µF as needed

2. Multilayer Ceramic Capacitors (MLCC)

Small SMD or through-hole rectangular components
May use 3-digit code (104) or letter code (475K)
Sometimes includes voltage code (104K250 = 0.1¬µF ¬±10% 25V)
Very small sizes may have no marking - check packaging

3. Electrolytic Capacitors

Usually printed directly (e.g., "100¬µF 25V")
No code decoding needed - value shown directly
CRITICAL: Note polarity marking (stripe = negative lead)
Tolerance typically ¬±20% (Z)

4. Film Capacitors

May use 3-digit code or direct printing
Often includes tolerance (J, K, M) and voltage rating
Example: 474K400 = 470nF ¬±10% 400V
Non-polarized - can be installed either way

5. SMD Capacitor Codes

Tiny surface mount capacitors (0402, 0603, 0805 sizes)
Often NO MARKING due to small size
Larger sizes (1206+) may have 3-digit code
Always verify with capacitance meter if unmarked

Step-by-Step Decoding Process

Complete Decoding Workflow:
Step 1: Identify capacitor type (ceramic, electrolytic, film)
Step 2: Locate the marking (may need magnification for small parts)
Step 3: Check if it's direct value (100¬µF) or code (104)
Step 4: If code, enter into decoder tool
Step 5: Note tolerance letter if present (J=¬±5%, K=¬±10%, M=¬±20%)
Step 6: Verify voltage rating is adequate for your circuit
Step 7: Check polarity if electrolytic/tantalum
Step 8: Use capacitance meter to verify if critical application

Ceramic capacitors lose capacitance under applied DC voltage

Safety Precautions & Critical Warnings

Voltage Rating Requirements

üî• Exceeding Voltage Rating = Fire/Explosion Risk

ALWAYS use voltage rating ‚â• 2√ó your maximum circuit voltage
Example: 12V circuit ‚Üí Use minimum 25V rated capacitor
AC circuits: Peak voltage = RMS √ó 1.414 (120VAC = 170V peak!)
Transients and spikes can exceed normal voltage
Ceramic caps can explode if over-volted (fragments are dangerous)
Electrolytic over-voltage causes venting, explosion, fire
Real Example - What NOT to Do:
‚ùå Using 10V capacitor in 9V circuit
‚Üí No safety margin for transients
‚Üí Battery "fresh" voltage can be 9.5-10V
‚Üí Inductive spikes can exceed 10V
‚Üí Result: Capacitor failure, possible fire
‚úì Correct: Use 16V or 25V rated capacitor

Polarity - Critical for Electrolytic/Tantalum

Identifying Polarity:

Negative lead: Marked with stripe, band, or arrows on body
Positive lead: Often longer leg (if not trimmed)
PCB markings: + symbol indicates positive pad
Reverse polarity = Explosion risk!

‚ö†Ô∏è Reverse Polarity Consequences:

Immediate failure (capacitor acts as short circuit)
Heating, venting of electrolyte gases
Possible explosion projecting hot material
Can damage other components from short circuit current
ALWAYS triple-check polarity before power-on!

Stored Charge & Discharge Safety

Capacitors Retain Charge After Power-Off!

Large capacitors (100¬µF+) can hold dangerous charge for minutes/hours
High voltage caps (>50V) require discharge before handling
Discharge method: Use resistor (1kŒ© - 10kŒ©, 1W+) across terminals
NEVER short terminals directly (arc/sparks dangerous)
Verify with voltmeter before touching
Safe Discharge Procedure:
Step 1: Power off circuit, disconnect from power source
Step 2: Wait 1 minute for natural discharge
Step 3: Use insulated tool to connect resistor across cap terminals
Step 4: Wait 5√ó RC time constant (calculate or use 30 seconds minimum)
Step 5: Verify with voltmeter reading <1V
Step 6: Now safe to handle

ESR & Power Dissipation

Overheating Risks:

High ripple current causes internal heating (I¬≤R losses)
Exceeding ripple current rating shortens life or causes failure
Signs: Bulging top, leaking electrolyte, hot to touch
Solution: Use capacitor with adequate ripple current rating
For power supplies: Check datasheet ripple specs carefully

General Handling Safety

Best Practices:

‚úì Always wear safety glasses when working with high-voltage circuits
‚úì Check voltage rating before installation
‚úì Verify polarity three times (measure, check PCB, check component)
‚úì Discharge large caps before touching

Troubleshooting Guide

Reading Difficulties

Problem: Markings Are Faded or Illegible

Solutions:
‚Ä¢ Use magnifying glass or phone camera with zoom
‚Ä¢ Try different lighting angles (side lighting reveals faint marks)
‚Ä¢ Gently clean with isopropyl alcohol (may reveal hidden markings)
‚Ä¢ Use capacitance meter to measure actual value
‚Ä¢ Check circuit schematic if available
‚Ä¢ Look for same capacitor elsewhere on PCB

Problem: SMD Capacitor Has No Marking

Solutions:
‚Ä¢ Very common for small SMD caps (0402, 0603 sizes)
‚Ä¢ Check PCB silkscreen for value label (C1, C2 near component)
‚Ä¢ Consult circuit schematic or BOM (Bill of Materials)
‚Ä¢ Use LCR meter or capacitance meter (must desolder first)
‚Ä¢ Common values: 0.1¬µF, 1¬µF, 10¬µF for power bypass

Code Interpretation Issues

Problem: Code Doesn't Make Sense (e.g., 999, 000)

Solutions:
‚Ä¢ May be date code or manufacturing code, not value
‚Ä¢ Look for other markings on same component
‚Ä¢ Some manufacturers use proprietary coding systems
‚Ä¢ Measure with meter to determine actual value
‚Ä¢ Search manufacturer datasheet using visible part numbers

Problem: Mixed Letters and Numbers (e.g., R47, 2A103)

Solutions:
‚Ä¢ "R" used as decimal point: R47 = 0.47 (unit depends on context)
‚Ä¢ Leading letters may indicate voltage: 2A103 = 100V, 103 code
‚Ä¢ Some film caps use different notation systems
‚Ä¢ Consult manufacturer datasheet for specific coding

Capacitor Identification

Problem: Don't Know What Type of Capacitor It Is

Visual Identification:
‚Ä¢ Ceramic: Small, disc or rectangular, tan/brown/blue color
‚Ä¢ Electrolytic: Cylindrical, aluminum can, polarity marking (stripe)
‚Ä¢ Tantalum: Small tear-drop or rectangular, often yellow/orange/black
‚Ä¢ Film: Box-shaped, often red/yellow/blue, larger size
‚Ä¢ MLCC: Tiny rectangular chip (SMD), ceramic material

Problem: Polarity Not Clear on Electrolytic

Identification Methods:
‚Ä¢ Stripe/band side = NEGATIVE terminal
‚Ä¢ Arrow markings point to NEGATIVE terminal
‚Ä¢ Longer lead = POSITIVE (if leads not trimmed)
‚Ä¢ Negative lead side often has chamfered edge on body
‚Ä¢ When in doubt: use voltmeter to check circuit polarity first

Measurement & Testing

Problem: Measured Value Doesn't Match Decoded Value

Possible Causes:
‚Ä¢ Normal tolerance: ¬±10% (K) or ¬±20% (M) is common
‚Ä¢ Capacitor must be desoldered for accurate measurement
‚Ä¢ In-circuit measurement affected by parallel components
‚Ä¢ DC bias reduces MLCC capacitance (measure at 0V bias)
‚Ä¢ Old electrolytic caps lose capacitance over time (aging)
‚Ä¢ Verify meter is set to correct range and zeroed

Problem: Capacitor Fails in Circuit

Diagnosis Steps:
1. Check voltage rating adequate (should be 2√ó circuit voltage)
2. Verify correct polarity for electrolytic/tantalum
3. Measure ESR (high ESR = failing electrolytic)
4. Check for physical damage (bulging, leaking, cracks)
5. Verify ripple current rating adequate for power supply use
6. Test with capacitance meter (look for open circuit or wrong value)
7. Replace with same or higher specs (voltage, ripple current)
