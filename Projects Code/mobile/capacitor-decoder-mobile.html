Capacitance

How to Use

• 3-digit code: First 2 digits = value, last digit = multiplier (number of zeros)
• 104 = 10 + 4 zeros = 100,000 pF = 100 nF = 0.1 µF
• 223 = 22 + 3 zeros = 22,000 pF = 22 nF
• 475 = 47 + 5 zeros = 4,700,000 pF = 4.7 µF
• 101 = 10 + 1 zero = 100 pF

Common Values

101 = 100 pF
102 = 1,000 pF = 1 nF
103 = 10 nF
104 = 100 nF = 0.1 µF
105 = 1 µF
106 = 10 µF
