Standard Resistor Values (E12)

  • 10Ω, 12Ω, 15Ω, 18Ω, 22Ω, 27Ω
  • 33Ω, 39Ω, 47Ω, 56Ω, 68Ω, 82Ω
  • And multiples: ×10, ×100, ×1k, ×10k, ×100k, ×1M

Common Capacitor Values

  • pF: 10, 22, 47, 100, 220, 470
  • nF: 1, 10, 100, 470
  • µF: 0.1, 1, 10, 22, 47, 100, 220, 470, 1000

Arduino Pin Limits

  • Max current per pin: 40 mA
  • Recommended per pin: 20 mA
  • Total for all pins: 200 mA
  • PWM pins (Uno): 3, 5, 6, 9, 10, 11
  • Analog pins (Uno): A0-A5

Ohm's Law

  • V = I × R (Voltage = Current × Resistance)
  • I = V / R (Current = Voltage / Resistance)
  • R = V / I (Resistance = Voltage / Current)
  • P = V × I (Power = Voltage × Current)
