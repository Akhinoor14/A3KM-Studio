    - script.m (Main MATLAB code) - results/ (PNG plots exported using saveas() or exportgraphics()) - documentation.pdf (Optional detailed report) - project.zip (Complete package with all files)
  2. 2
    Create Project Folder
    Navigate to: Projects Storage/MATLAB Projects/
    Create a new folder with a unique ID (e.g., "matlab-solar-pv-01")
  3. 3
    Upload Files via GitHub
    Upload files to the project folder:
    matlab-solar-pv-01/ ├── solar_pv_analysis.m ├── results/ │ ├── iv_curve.png │ ├── pv_curve.png │ └── efficiency.png ├── documentation.pdf └── solar_pv_analysis.zip

    Add project entry to the JSON editor below. Follow the template structure for consistency.
