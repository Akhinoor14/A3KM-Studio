  • Formats: Use .jpg, .png, or .svg

  • File Names: Use lowercase with hyphens (e.g., my-arduino-project.jpg)

  1. Go to your GitHub repository
    2. 
                    
  2. Navigate to: images/blog/

    6. 
                    
  6. Click "Commit changes"
    7. 
                


                
Step 3: Use Image in Blog Post

                

                    For Cover Image:
                    
images/blog/my-post-cover.jpg

                


                

                    In Markdown Content:
